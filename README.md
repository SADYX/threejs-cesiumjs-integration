# threejs-cesiumjs-integration

一个集成了 Threejs 和 Ceium 的示例，包括了

-   射线检测（THREE.Object3D / CESIUM.Entity）

-   飞行至模型

-   贴地坐标转化

# 快速启动

```bash
git clone
```

```bash
npm install
```

```bash
npm run dev
```
